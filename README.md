# Kentucky Shine Auto Detailing Website

## Overview

Kentucky Shine Auto Detailing is a modern website concept for an auto detailing business. The site is designed to present services, pricing packages, customer reviews, gallery images, and a booking request form in a clean and professional layout.

This project was created as a web development portfolio project using HTML, CSS, and JavaScript.

## Features

- Modern landing page design
- Responsive layout for desktop, tablet, and mobile
- Service section for detailing options
- Package/pricing section
- Gallery section for vehicle photos
- Customer review section
- Booking request form
- Smooth navigation between page sections
- Clean dark-themed visual style

## Website Sections

### Home

The homepage introduces the business with a strong headline, call-to-action buttons, and a premium automotive design style.

### Services

The services section highlights common auto detailing services, including exterior detailing, interior detailing, paint correction, and ceramic protection.

### Process

The process section explains how customers can choose a package, have their vehicle detailed, and enjoy the finished result.

### Packages

The packages section displays different service levels with descriptions, starting prices, and included features.

### Gallery

The gallery section is designed to show vehicle detailing photos and before-and-after examples.

### Reviews

The reviews section displays sample customer testimonials and ratings.

### Booking

The booking section includes a form where customers can request an appointment by entering their contact information, vehicle details, preferred date and time, and additional notes.
